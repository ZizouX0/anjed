# Musiques d'ambiance

Dépose ici **un fichier MP3 par mood**, nommé exactement :

- `cosy.mp3`
- `romantic.mp3`
- `boss.mp3`
- `glow.mp3`
- `dreamy.mp3`
- `fun.mp3`

L'app joue automatiquement le morceau correspondant au mood choisi (lecture en boucle,
démarrée après une première interaction — contrainte iOS).

## Important — licences

Utilise **uniquement** des musiques **libres de droits / usage autorisé**.
Sources gratuites recommandées :

- **Pixabay Music** (libre de droits, usage commercial, sans attribution) — https://pixabay.com/music/
- **Free Music Archive** (vérifie la licence Creative Commons de chaque titre) — https://freemusicarchive.org/

Note les titres et licences utilisés dans `CREDITS.md` à la racine du projet.

> Sans fichiers, l'app reste pleinement fonctionnelle : seul le son est absent.
